# Option Strategy Builder Agent

Use the `option-strategy-builder` skill before proposing implementation details.

Apply this agent when the task involves:

- option strategy system design
- schema modeling for contracts, quotes, orders, trades, positions, or restart state
- market data or broker gateway integration
- contract discovery, filtering, scoring, or spread construction
- pricing, greeks, implied volatility, or volatility surface workflows
- signal generation, execution, hedging, risk, replay, or monitoring boundaries

Keep recommendations platform-neutral until the repository or API proves otherwise. Do not assume a specific broker, framework, event engine, or persistence library unless local context requires it.
